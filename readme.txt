Cocorico Shortcodes adds a useful set of shortcodes to enhance your WordPress powered website. Our shortcodes are fully internationalized.

== Description ==

Cocorico Shortcodes gives you the ability to enhance your website content with the following shortcodes :

* Columns for better rendering (2, 3 or 4 columns combinations)
* Message boxes to catch visitor attention (4 styles available)
* Buttons to influence people to click (10 styles available)
* Tabs to display the right content at the right time
* Separators to delimitate your content (4 styles available)
* Toggles to create FAQs
* Call to actions to make them click

If you don't use WordPress in english, you're gonna love this plugin because **all our shortcodes are internationalized** (names and attributes). Just grab PoEdit and translate the po file and you'll be set.

English shortcodes tends to confuse customers. With Cocorico Shortcodes, they will enhance their content easily. 

For example, the following shortcode :

*[cocorico_button url="http://wordpress.org" target="current_window" size="medium" color="blue"]WordPress Power[/cocorico_button]*

in french becomes :

*[cocorico_bouton url="http://wordpress.org" cible="meme_fenetre" taille="normal" couleur="bleu"]WordPress est génial[/cocorico_bouton]*

Avec Cocorico Shortcodes, les shortcodes apparaitrons comme moins obscurs pour vos clients.

== Installation ==

= English =

1. Upload the `cocorico-shortcodes` folder to the `/wp-content/plugins/` directory
1. Activate the Cocorico Shortcodes plugin through the 'Plugins' menu in WordPress
1. Use the dropdown that appeared in the visual editor to insert your first shortcodes

= Français =

1. Envoyez le dossier `cocorico-shortcodes` dans le dossier `/wp-content/plugins/`
1. Activez Cocorico Shortcodes sur la page des plugins de WordPress
1. Utilisez le menu déroulant qui est apparu dans l'éditeur visuel pour insérer vos premiers shortcodes
